# 📄 Documentation

This folder contains supporting documentation for the **Inventory Intelligence Dashboard** project.

## Contents
- `dax-measures.md` → Key DAX measures used in the dashboard
- `assumptions.md` → Business and reporting assumptions applied in the dashboard
- `insights-summary.md` → Summary of key insights and business interpretation
- `data-model.png` → Data model view from Power BI

## Purpose
These files provide additional context on the dashboard logic, modelling approach, and reporting structure used in the project.

## Note
This repository contains a portfolio version of the project. Any confidential or identifying business information has been excluded.
